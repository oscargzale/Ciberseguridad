Sistema de Gestión Académica en MySQL

Este proyecto corresponde al trabajo final de la asignatura Base de Datos.

Incluye:
- Creación de tablas con claves primarias y foráneas
- Inserción de datos
- Consultas básicas
- Uso de JOIN
- Estadísticas
- Manipulación de datos (INSERT, UPDATE y DELETE)

El proyecto fue desarrollado y probado en MySQL Workbench.
